# bp-react-webpack
Simple structured staring point for react projects, with webpack.

## Features

* React 16
* Webpack 4
* Babel
* Staging ES Next Features
* Eslint
* SASS

## Installation

* `git clone git@github.com:jghlt/bp-react-webpack.git`
* cd bp-react-webpack
* npm install
* npm start
* visit `http://localhost:3000/`
* npm run build
